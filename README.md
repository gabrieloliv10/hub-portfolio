# hub-portfolio

Este repositório atua como o hub central para a organização, versionamento e apresentação dos meus projetos de tecnologia. Aqui estão reunidas as aplicações desenvolvidas como parte da minha jornada profissional, abrangendo diversas áreas e stacks conforme aplico soluções a problemas reais.

## 🚀 Objetivo
Centralizar o acesso ao meu portfólio, permitindo uma visão organizada do meu aprendizado contínuo e das entregas técnicas que venho realizando. Este é o ponto de partida para quem deseja explorar o que estou desenvolvendo.

## 🛠️ Stack Tecnológica
*Esta seção será atualizada conforme a evolução dos projetos integrados ao hub.*

* **Linguagens**: Python, SQL, JavaScript, HTML, CSS.
* **Ferramentas**: VS Code, Git, Power BI, SharePoint.
* **Deploy**: Vercel.
